Web Applications (242-001-S2024)

Firstly, students will utilize HTML and CSS to design the front-end of a website; matching a given specification. Secondly, they will use JavaScript to add functionality to their website. Lastly, they will learn to utilize node and server side JavaScript to store and retrieve data from a server.

Assignments:

    Assignment 1 - Basic HTML

    This assignment had us create a webpage and taught us how to use gitHub and VS Code. We had to create text of different sizes, a bulleted list, add images and links. We made the website on a topic of our choice.

    Assignment 2 - Basic CSS

    This assignment had us create a webpage that implemented basic styling to be more presentable to users. We had to change the background color, text font, and add a main navigation to the top of the page.

    Assignment 3 - Page Layout

    This assignment had us create a webpage that changes based on the the platform of viewing. We did this by creating columns for big screens. Sectioning the page helps the viewer focus more.

    Assignment 5 - Recreate CSS Page

    This assignment had us re-create a webpage from the UofSC website that needed to change based on size of the screen.

    Assignment 7 - Javascript, Buttons, Functions

    This assignment had us use html,css,and JavaScript to create an interactable website with three different functions.

    Assignment 8 - Conditionals

    This assignment we had to use if-statements in java script to display different images to the user based on their input.

    Assignment 9 - For Loops

    This assignment had to use a setInterval and for loops to iterate through a timer and through a list of li elements.

    Assignment 10 - Arrays

    This assignment had to use a setInterval to run through a list of advertisement quotes. Also, we had to loop through an arrayList of Objects that stored an image and its attribution.

    Assignment 11 - Classes

    This assignment used an array of classes to populate the DOM with each section being clickable to add a modal screen to display the additional information.

    Assignment 12 - Parsing JSON

    This assignment I had to connect the code to a JSON file via internet link. Then parse through the JSON file in order to extract the data. The separate sections were tricky.
